# 🧱 Retro Gameboy Tetris (Streamlit)

A mobile-responsive Tetris clone built with **Python**, **Streamlit**, and a dash of **JavaScript**.

## 🚀 Features
- **Responsive Design:** Detects if you are on a phone or PC.
- **Gameboy UI:** Styled with CSS to look like a vintage handheld.
- **Live Deployment:** Ready to be hosted on Streamlit Community Cloud.
